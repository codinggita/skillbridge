# 🚀 Local Skill Exchange & Micro-Service Marketplace

## 📌 Overview

Local Skill Exchange & Micro-Service Marketplace is a full stack web application that enables users to showcase their skills and connect with nearby people who require small services.
The platform helps individuals offer micro-services such as tutoring, repairs, design work, fitness guidance, and other local services in a simple and accessible way.

This application aims to promote community collaboration, improve skill utilization, and create earning opportunities for students and local professionals.

---

## 🎯 Problem Statement

In many small towns, colleges, and local communities, people possess valuable practical skills but lack a proper digital platform to offer their services.
At the same time, users struggle to find reliable and affordable local help for small tasks.

Existing large service platforms are often:

* Focused on big cities
* Complicated to use
* Not suitable for micro-level services
* Expensive or commission-heavy

This project proposes a simple and responsive web platform where users can list their skills, search for nearby services, and manage service bookings efficiently.

## ✨ Core Features

### 🔐 Authentication System

* User Signup
* User Login
* Password validation
* Protected routes

### 👤 User Profile Management

* Add personal details
* Add skills and experience level
* Profile image support
* Availability status

### 📢 Service Listings

* Users can post services with:

  * Service title
  * Description
  * Category
  * Price
  * Location

### 📅 Booking System

* Users can request services
* Service provider can accept or reject requests
* Booking status tracking

### 🔎 Search, Filtering & Sorting

* Search services by skill or keyword
* Filter by category or price range
* Sort by price, rating, or latest

### ⭐ Rating & Feedback

* Users can rate service providers
* Basic feedback system after service completion

### 📄 Pagination

* Backend pagination using MongoDB `limit` and `skip`
* Frontend pagination UI for service listings

### ⚡ Debounced Search

* Optimized API calls while typing

### 🌍 Global State Management

* Authentication state
* Theme preference
* Booking and service state

### 🎨 Theme Support

* Dark Mode / Light Mode toggle
* Persist theme using LocalStorage

### 📱 Responsive Design

* Mobile friendly
* Tablet friendly
* Desktop friendly

### ❗ Error Handling

* Proper backend API error responses
* Frontend error alerts
* Loading indicators during API calls
